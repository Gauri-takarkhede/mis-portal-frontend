🎓 Student MIS Portal (Java Full Stack)

A production-grade role-based Student Management System built using Java Full Stack technologies (Spring Boot, Angular, PostgreSQL).
The application follows a scalable layered architecture and is deployed in a cloud environment with secure authentication and role-based access control (RBAC).

This system helps educational institutions manage student records, elective allocation, bonafide workflows, and scholarships through a centralized web portal.

🏗 Architecture & Design

The project follows a production-ready layered architecture.

Backend (Spring Boot)

Controller Layer – Handles REST API requests

Service Layer – Business logic implementation

Repository Layer – Database interaction using Spring Data JPA

Entity Layer – Database models

Frontend (Angular)

Component-driven architecture

Lazy-loaded modules for improved performance

Modular UI structure using Angular Material and Bootstrap

Other Design Features

RESTful API design

Centralized exception handling

Environment-based configuration

Production-ready CORS configuration
